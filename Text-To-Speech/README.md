# Text-To-Speech
A program that can convert Text into Speech using python


# Dependencies:

*pyttsx3*
```python

pip install pyttsx3
```
*pyaudio*
```python
pip install pyaudio
```


# Run:
*Add the text in the input.txt file*

```
python text-to-speech.py
```
